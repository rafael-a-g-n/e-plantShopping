This project, originally forked from `ibm-developer-skills-network/e-plantShopping:main`, delivers a dynamic and intuitive e-commerce experience for plant enthusiasts. ## âš™ï¸ Installation & Setup

To get this project up and running locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rafael-a-g-n/e-plantShopping.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd e-plantShopping
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```
    The application will be accessible at `http://localhost:5173` (or another port if 5173 is in use).
